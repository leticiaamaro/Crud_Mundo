# CRUD Mundo

Este projeto é um sistema de CRUD para países e cidades do mundo, focado no gerenciamento de dados geográficos. O objetivo é permitir o cadastro, edição, visualização e exclusão de informações sobre países e cidades.

## Estrutura do Projeto
- **web/**: Aplicação web (Front End)
- **database/**: Scripts e arquivos do banco de dados

## Tecnologias Utilizadas
- **Front End**: HTML, CSS, JavaScript
- **Back End**: PHP
- **Banco de Dados**: MySQL

## Funcionalidades
- Cadastro, edição, exclusão e visualização de países
- Cadastro, edição, exclusão e visualização de cidades
- Relacionamento entre países e cidades

## Como executar
1. Configure o banco de dados MySQL usando os scripts em `database/`
2. Implemente o Back End em PHP para manipulação dos dados
3. Desenvolva o Front End em HTML, CSS e JavaScript na pasta `web/`

## Objetivo
Facilitar o gerenciamento de dados geográficos, promovendo o aprendizado de tecnologias web e banco de dados relacional.
